注册表 - 风格参考

接口参数说明:

参数 是否必须 参数类型 说明
name 是 string 姓名
telephone 是 int 电话
sex 否 string 性别： 0保密，1男，2女
QQ 否 string QQ
level 否 string 层次
major 否 string 专业
pageType 否 string 页面类型： 表单要注册信息。如：成考，自考，也可以是页面的title
location 否 string 所在地区。 默认自动获取， 可以自定义
url 否 string 来源URL。 默认自动获取
comment 否 string 备注

注1：表单自定义验证规则: 

1. 表单form： class = CCForm
2. 表单验证：
    a) 在需要验证的表输入框类上添加 class="required", 
    b) 规则：data-valid="isNonEmpty"
    c) 提示信息：data-tip="请输入您的姓名"， 不正确提示信息：data-error="用户名不能为空"
    d) 提示信息显示位子：需在要输入框外加div. 类名：cc_item， 可以此类cc_item 设置显示属性，如：字体大小
规则 说明
isNonEmpty 不能为空
isMobile 不能为空
isEmail 是否为邮箱
minLength 要大于
maxLength 要小于
between 大于小于
onlyEn 纯英文
onlyZh 纯中文
onlyNum 数字包含小数
onlyInt 整数
isChecked 选择：单选，多选

版本：v1.1 与 v1.2 区别

在v1.1 的基础上增加动态加载样式文件， 回调(提交成功后执行)

v1.2 插件的引用：
<script src="http://post.aihuagrp.com/assets/js/libs/cc/jquery.CCForm.v2.js"></script>

提交注册式表单使用：版本1.2(推荐)

1. 按规则写from 表单， 参考一般写法(用户名，手机号， 提交按钮)
<form class="CCForm"> 
    <div class="cc_item">
        <label><span>*</span>用户名</label>
        <input type="text" name="name" class="required input-text radius"  data-tip="请输入您的姓名" data-valid="isNonEmpty||maxLength:2" data-error="用户名不能为空||长度不能小于2位" placeholder="用户名">
    </div>

    <div class="cc_item">
        <label><span>*</span>手机号</label>
        <input type="text" name="telephone" maxlength="11" class="required input-text radius" value="" data-tip="请输入您的手机号" data-valid="isNonEmpty||isMobile" data-error="手机号不能为空||手机号格式不正确" placeholder="手机号">
    </div>

    <div class="cc_item">                
        <input type="hidden" name="pageType" value="成考-百度渠道">          
        <input class="btn btn-primary" type="submit" value="提交">
    </div>
</form>

<script src="http://post.aihuagrp.com/assets/admin/js/jquery.min.js"></script>
2. 引入JS，调用-提交。 注意依赖 jquery，需引入
<script src="http://post.aihuagrp.com/assets/js/libs/cc/jquery.CCForm.v2.js"></script>
<script>$.CCForm();</script>


说明：对与不同项目只需要指定不同的提交URL 地址一个参数就行了
<script>$.CCForm({url:'http://post.aihuagrp.com/email【各项目标识名】'});</script>

调用方式：

调用 方式1：无参数
$.CCForm();


调用 方式2：有参数
$.CCForm({
    form:'.CCForm',                                         // 可选。表单类名(class 要带点)，默认为CCFrom
    url:'http://www.xxx.com/post',                          // 可选。要提交的url(注：区分大小写)
    jumpUrl:'http://www.xxx.com',                           // 可选。提交成功后跳转置页面
    callback:null,                                          // 可选。提交成功后执行. v2 添加
    successMsg:'提交成功<br>我们的工作人员会尽快与您联系',  // 可选。提交成功后，提示的信息
    errorMsg:'提交失败',                                     // 可选。提交失败后，提示的信息
    delay:10000                                             // 可选。提交后提示信息框的延时显示时间
});


注意： 方式2中url 参数是区分大小写，写错提交不成功


// 测试用
$.CCForm({
	url:"http://xxx.com/email",
	callback:function(){
		console.log('执行正确后调用')
	}
})
